## Getting Started

To get started with the project, follow these steps:

1. Clone the repository to your local machine:
```git clone <repo-url>```

2. Navigate to the project directory:
```cd client```
or
```cd server```

4. Install dependencies using npm or yarn:
```npm i```
or
```yarn```
 
5. Run the project:
```npm run dev```
or
```yarn dev ```
